# Ursa Infra

> Deploy and run an ursa node, with an nginx proxy infront. Setup with cerbot and letencrypt. Certs are auto renewed. 

## Prerequisites

- Install [Docker](https://docs.docker.com/get-docker/)

## Usage

```sh
cd ursa

docker build -t ursa -f ./Dockerfile .
```
